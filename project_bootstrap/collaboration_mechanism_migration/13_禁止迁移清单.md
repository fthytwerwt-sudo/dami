# 禁止迁移清单

以下内容标记为 `not_migrated`：

## 领域专属机制

- 视频生产、素材、镜头、字幕、剪辑、卡片、审片包、TTS、音色、画幅、发布候选等专属流程和状态。
- 旧项目的人物、产品、文案、客户、运营目标和业务判断。
- 旧项目的 `technical_preview`、`publish_candidate`、`send_ready` 等业务状态名称。

## 固定技术实现

- DashVector、Chroma 或任何单一 RAG 技术作为强制依赖。
- DeepSeek 或任何外部 AI 每轮强制参与。
- 旧仓库 API、脚本、schema、fixture 和验证器的直接复制。
- 只适用于旧项目的媒体、生成、审美或发布验证。

## 路径与历史

- 旧项目绝对本地路径。
- 历史日志中的旧结论、旧状态和已失效路线。
- 旧 GPT Project 上传包中的动态业务内容。
- 大型媒体、缓存、构建产物、secret、Token、Cookie、认证信息。

## 新项目禁止自动写入

- 未经用户确认的国家、市场、平台、商业模式和渠道结论。
- 暂时价格、客户资料、短期任务状态和一次性研究结果进入静态指令。
- 既有 `foreign_trade_low_gi_plan/` 的确定性表述自动升级为正式事实。

## 保留方式

可保留“为什么未迁移”的证据索引与摘要；不得复制大量原文，不得让被拒内容成为默认入口。
