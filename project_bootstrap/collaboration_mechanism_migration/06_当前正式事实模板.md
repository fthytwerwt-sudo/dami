# 当前正式事实模板

> 本文件仅为模板。所有栏目默认空，不代表任何国家、产品、渠道或商业模式已经确认。

元数据：

- `document_status`: `template_only`
- `last_updated_at`:
- `updated_by`:
- `reviewed_by`:
- `evidence_cutoff_at`:

## 已确认事实

每条使用：`fact_id / statement / scope / evidence_source / verified_at / valid_until / confirmed_by / supersedes / status`。

## 有证据但未最终确认

每条补充：`missing_confirmation / risk_if_promoted_early`。

## 候选方向

每条补充：`hypothesis / evaluation_criteria / next_validation / owner`。

## 已否决方向

每条补充：`rejected_reason / evidence / rejected_at / reconsider_if`。

## 当前实验

每条补充：`experiment_id / hypothesis / permitted_actions / forbidden_actions / success_metric / failure_metric / start_at / stop_at / status`。

## 当前阻断

每条补充：`blocker_id / affected_scope / evidence / unblock_condition / owner / next_review_at`。

## 等待人工决策

每条补充：`decision_id / options / tradeoffs / deadline / decision_authority / default_safe_action`。

## 证据来源登记

每条补充：`source_id / source_type / title / issuer / acquired_at / effective_at / valid_until / storage_path_or_url / integrity_check / limitations`。

## 被替代关系

每条补充：`old_fact_id / new_fact_id / replacement_reason / effective_at / approved_by`。

## 禁止事项

- 不从聊天、旧项目或候选研究自动填入。
- 不把计划写成事实。
- 不把文件新旧当作唯一权威标准。
- 没有证据、有效期和确认者的条目不得进入“已确认事实”。
